Commandes de base pour Screen

Action Commande
Démarrer une nouvelle session screen screen
Démarrer une session avec un nom spécifique screen -S session_name
Lister toutes les sessions screen screen -ls
Attacher à une session existante screen -r [session_id]
Détacher d'une session screen Ctrl-a puis d
Fermer une session screen exit ou Ctrl-d
Créer une nouvelle fenêtre à l'intérieur d'une session screen Ctrl-a puis c
Naviguer entre les fenêtres Ctrl-a puis n (prochaine) / Ctrl-a puis p (précédente)
Diviser la vue actuelle Horizontalement : Ctrl-a puis |
Verticalement : Ctrl-a puis S
Fermer une session screen mais la laisser tourner en arrière-plan Ctrl-a puis d
Fermer une session scren depuis une commande sans se tattacher à la session screen -XS session-id quit
